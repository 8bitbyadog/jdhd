# ADHD Quest: JRPG-Style Productivity App

A productivity web app designed specifically for people with ADHD, featuring a cute but cool JRPG-style UI. This app helps manage tasks, track time, and capture creative ideas while providing the right balance of flexibility and structure for creative goals.

## Features

### Task Management (Quest Log)
- Create and manage tasks as "quests"
- Set priority levels and categories
- Track completion status
- JRPG-inspired visual design

### Time Tracking (Time Crystal)
- Pomodoro-style timer with focus and break periods
- Visual time tracking
- Session history
- Sound notifications

### Creative Tools (Idea Inventory)
- Capture creative ideas and inspirations
- Random creative prompts
- Categorize and expand on ideas
- Visual card-based interface

### No Account Required
- All data stored locally in your browser
- Export/import functionality for data backup
- No sign-up or login needed

## Embedding in Google Sites

This app is designed to be embedded in Google Sites using the HTML embed feature. See the [embedding instructions](EMBEDDING.md) for details.

## JRPG-Style UI

The interface features:
- Pixel-inspired design elements
- JRPG-style menus and panels
- Game-like interactions
- Responsive design that works on all devices

## For ADHD Users

This app was specifically designed for ADHD users with features that:
- Reduce overwhelm with clear visual hierarchy
- Provide just enough structure without being rigid
- Use visual cues and feedback to maintain engagement
- Support both hyperfocus and task-switching
- Help maintain creative momentum

## For Creative Practice

The app supports creative workflows by:
- Capturing ideas when inspiration strikes
- Providing creative prompts to overcome blocks
- Balancing flexibility with helpful limitations
- Tracking progress on creative projects
- Supporting both structured and unstructured creative time

## Technical Details

- Built with Svelte as a web component
- No server dependencies - runs entirely in the browser
- Lightweight and fast
- Accessible design

## Development

### Prerequisites
- Node.js and npm

### Setup
```bash
# Install dependencies
npm install

# Build the app
npm run build

# Run development server
npm run dev
```
